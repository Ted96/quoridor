Quoridor game in C
_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-

Run with:
gcc -o <out> quoridor.c
./<out>

To show available commands / help :  type "list_commands" in game
